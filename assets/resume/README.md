# Resume Setup Guide

This folder is where you should place your resume PDF file.

## Instructions:

1. **Add Your Resume PDF**
   - Save your resume as: `paul-owoeye-resume.pdf`
   - Place it in this folder: `assets/resume/`
   - The file will automatically be available for download and viewing on your portfolio page

2. **Resume File Settings**
   - Filename: `paul-owoeye-resume.pdf` (or update the filename in `portfolio.html`)
   - Format: PDF (recommended for better compatibility)
   - Location: `assets/resume/paul-owoeye-resume.pdf`

3. **Update the Portfolio Page (Optional)**
   - If you want to use a different filename, edit `portfolio.html`
   - Find the line: `<a href="assets/resume/paul-owoeye-resume.pdf"`
   - Replace `paul-owoeye-resume.pdf` with your filename

4. **Features Included**
   ✅ Download PDF button
   ✅ View in browser button
   ✅ Professional resume display section
   ✅ Work experience timeline
   ✅ Skills and expertise showcase
   ✅ Education section
   ✅ Certifications

5. **Resume Content Sections**
   The portfolio page includes:
   - Professional summary
   - Work experience with descriptions
   - Education background
   - Skills organized by category
   - Certifications and achievements
   - Contact information

## Next Steps:
1. Create your resume PDF using any tool (Word, Google Docs, Canva, etc.)
2. Save it as `paul-owoeye-resume.pdf`
3. Move it to this folder: `assets/resume/`
4. Visit `portfolio.html` to see it displayed with download options

Enjoy your new portfolio page! 🚀
